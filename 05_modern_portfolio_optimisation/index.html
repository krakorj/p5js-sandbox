P5 script - Modern portfolio theory

Modern portfolio theory (MPT) , or mean-variance analysis, is a mathematical framework for assembling a portfolio of assets such that the expected return is maximized for a given level of risk

MPT is a task for Monte-Carlo method to get optimal solution fast.

Info! To run this p5.js project use a local server. For python3 installed use python -m http.server and click http://localhost:8000
